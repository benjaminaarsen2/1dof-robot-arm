# 1-DOF 机械臂用户指南

欢迎！本指南介绍如何通过串口命令控制 1-DOF（单自由度）机械臂。

## 连接机械臂

1.  使用提供的 USB 数据线将机械臂连接到电脑。
2.  打开串口监视器应用（如 Arduino IDE 串口监视器、PuTTY 或 PlatformIO 的串口监视器）。
3.  按以下参数配置串口连接：
    *   **端口**：分配给机械臂的 COM 或 TTY 端口
    *   **波特率**：`115200`
    *   **换行符**：换行（`\n`）

连接成功后，机械臂会发送欢迎信息。

## 控制命令

通过串口监视器输入以下命令并回车，即可控制机械臂。

### 1. 位置移动

让机械臂立即移动到指定角度。

*   **命令**：`P<angle>`
*   **`<angle>`**：目标角度，整数（0 到 180）
*   **示例**：`P90` - 机械臂将移动到 90 度位置

### 2. 定时位置移动

让机械臂在指定时间内平滑移动到目标角度。

*   **命令**：`T<angle>,<duration>`
*   **`<angle>`**：目标角度（0-180）
*   **`<duration>`**：移动所需时间，单位为毫秒（如 1000ms = 1 秒）
*   **示例**：`T180,2000` - 机械臂将在 2 秒内移动到 180 度

## 反馈信息说明

机械臂会实时反馈状态信息。每条信息前缀表示其类型：

*   **`[INFO]`**：一般信息，如开始移动时提示
    *   *示例*：`[INFO] Starting move from 90 to 180 in 2000ms`

*   **`[SUCCESS]`**：命令成功完成的确认
    *   *示例*：`[SUCCESS] Movement complete.`
    *   *示例*：`[SUCCESS] Moved to position: 90`

*   **`[ERROR]`**：发送无效或无法识别命令时出现
    *   *示例*：`[ERROR] Unknown command: X123`
    *   *示例*：`[ERROR] Invalid format. Use T<angle>,<time>`
